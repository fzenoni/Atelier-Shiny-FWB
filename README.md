[Florian Zenoni](https://www.linkedin.com/in/florianzenoni/) du Data Office (florian.zenoni@cfwb.be)

## Introduction et objectifs

Ce dépôt contient le matériel utilisé dans l'atelier R Shiny proposé aux membres du Ministère de la Fédération Wallonie-Bruxelles. Cet atelier a pour but de vous aider à débuter dans la construction d'applications Shiny en R, si vous possédez déjà des compétences de bases en programmation avec R.

Cet atelier a pour objectifs :
* de vous guider dans la paramétrisation d'un projet R,
* de vous permettre de coder votre première application Shiny,
* de vous apprendre les bases de l'interface utilisateur (UI) et du back-end,
* de vous enseigner les bases de la réactivité et d'optimiser un app simple

## Organisation du dépôt

### Setup

**Démarrage rapide** : Utilisez les liens shinylive fournis dans le README pour chaque exercise, car ils ne nécessitent d'aucun setup.

* Partie 2 - Syntaxe de réactivité

Chaque partie a un dossier dédié dans ce dépôt :
